## 搬瓦工购买后第一步：找到三样信息

收到确认邮件之后，先别急着操作，去找这三样东西：

1. **IP 地址**（Public IP address）
2. **SSH 端口**（SSH Port，搬瓦工不是默认 22，是随机生成的五位数端口）
3. **root 密码**

这三样东西才能让你连上服务器。

问题是，搬瓦工现在买完不会直接把密码发给你。邮件里有 IP 和端口，密码需要自己去 **KiwiVM 后台**手动重置。具体位置：登录后台 → 找到你买的 VPS → 点击 KiwiVM Control Panel → 左侧菜单选 "Root password modification" → 点 "Generate and set new root password"。

**注意一个容易踩的坑**：重置密码的时候 VPS 必须处于开机状态，不然会报错（错误代码 739102 就是这个原因）。生成新密码后，那个密码只显示一次，务必马上复制保存。

## 怎么连上 VPS：SSH 客户端选哪个

有了 IP、端口、密码，下一步是选一个 SSH 客户端把这三样填进去，然后连上去。

**Windows 用户**推荐用 Xshell（官方有免费家庭版）或者 FinalShell（界面友好，文件传输也方便）。新建连接的时候注意：主机填 IP，端口填 KiwiVM 里看到的那个随机端口（不是 22），用户名填 root，密码填刚才重置的那个。

**Mac 用户**直接开终端，不需要另装软件，命令格式是这样的：

bash
ssh root@你的IP地址 -p 你的SSH端口


比如 `ssh root@123.45.67.89 -p 28975`，然后输入密码就行。

**手机或者紧急情况**下，直接用 KiwiVM 里的 Root shell – interactive，浏览器里直接跑，不需要任何客户端。

连上去之后，你会看到一个命令行界面。到这一步，说明你已经成功远程控制了这台服务器。

## 常见新手问题（FAQ）

**Q：买完搬瓦工，KiwiVM 面板在哪里登录？**
A：用你购买时注册的邮箱和密码登录搬瓦工账号后台，进入 Services → My Services，找到你的 VPS，点那个 "KiwiVM Control Panel" 按钮就进去了。

**Q：SSH 密码输入正确但连不上，是什么问题？**
A：几种可能：一是端口没填对（搬瓦工 SSH 端口不是 22）；二是 VPS 没开机；三是 IP 被墙，需要检测一下连通性再排查。SSH 能 Ping 通但连不上，多半是服务配置或防火墙的问题。

**Q：流量用完了会怎样？**
A：流量超额后速度会被限制，不会直接断线。等下个计费周期流量会自动重置。急用可以在 KiwiVM 后台花费额外费用购买额外流量包。

**Q：搬瓦工支持哪些付款方式？**
A：支付宝、PayPal、信用卡、银联都支持，对国内用户来说支付宝是最方便的选项。

**Q：买了之后不满意可以退款吗？**
A：支持 30 天内全额退款，退款申请通过工单提交。用了 28 天觉得不合适去提申请，只要在期限内一般都能退。

**Q：KiwiVM 里 "Root shell – basic" 和 "Root shell – interactive" 有什么区别？**
A：basic 是非交互式，适合执行简单命令；interactive 是完整的交互式 shell，类似在浏览器里开了个终端，更接近 SSH 连接的体验。
